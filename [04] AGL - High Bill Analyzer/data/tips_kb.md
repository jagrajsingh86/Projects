# Energy Saving Tips (Synthetic KB)
- Heating: Set thermostat between 18–20°C in winter; check for drafts; service units.
- Cooling: Use fans first; set AC to 24–26°C in summer.
- Hot water: Reduce shower time; insulate pipes.
- Appliances: Avoid standby; run washing at cold when possible; schedule dishwasher off-peak.
- Solar: Shift heavy loads to daylight; check inverter status.
